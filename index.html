기여한 부분과 트러블 슈팅
건강기능식품 리뷰 페이지 구현

  • 작성 폼에서 DB에 등록된 제품을 검색하는 기능
    • 목적
      • 성분명, 제품명을 검색하여 정확한 제품 정보를 전달

    • 트러블 슈팅
      • Supabase의 like 메서드를 적용

  • 검색 결과 모달에서 제품 이미지 리스트와 성분명 보여주기
    • 목적
      • 이미지에 마우스 오버하면 기능을 보여줘서 추가 정보를 제공
      • 3줄을 초과하는 기능은 초과분을 숨기고 ‘더 보기’를 누르면 전체 출력
    • 문제
      • ‘더 보기’ 클릭 시 검색 모달창이 닫히고 연결된 이벤트가 실행되는 event bubbling 발생
    • 트러블 슈팅
      • stopPropagation으로 해당 이벤트만 실행되도록 수정
